# GitFetch
JavaScript application that fetches data from the GitHub REST API v3 and displays it on the web-page.

## Getting Started
Start by either downloading the zip file or clone with HTTPS.

### Prerequisites
* Node.js & NPM (https://nodejs.org/en/)
